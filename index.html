Exercício 1

O programa lê as medidas dos lados de um triângulo e determina sua classificação com base nessas medidas. Se todos os lados têm o mesmo tamanho, o triângulo é classificado como equilátero. Se dois lados têm o mesmo tamanho e o terceiro é diferente, é classificado como isósceles. Por fim, se todos os lados têm tamanhos diferentes, o triângulo é escaleno.

Exercício 2

O programa calcula o valor total a ser pago por uma pessoa após a aplicação de descontos. Ele solicita o valor total da compra e um código que identifica o tipo de cliente: comum (1), funcionário (2) ou VIP (3). Com base no código fornecido, o programa aplica um desconto de 10% para funcionários e 5% para clientes VIPs. A saída do programa exibe o valor total da compra após o processamento e indica a categoria do desconto aplicado, se houver.

Exercício 3

O programa desenvolvido recebe como entrada os gols marcados por cada time em um jogo de futebol e determina o resultado final, informando se foi um empate, vitória do primeiro time ou vitória do segundo time. Ele realiza essa verificação comparando os números de gols marcados por cada equipe e exibindo o resultado correspondente com base nessa comparação. Assim, o usuário pode rapidamente saber o desfecho do jogo apenas fornecendo o placar final.

Exercício 4

Um programa desenvolvido para armazenar os nomes de 10 pessoas em um vetor. Após a entrada dos nomes, o programa permite a leitura de mais um nome para efetuar uma busca entre os nomes inseridos anteriormente. Se o nome fornecido estiver presente no vetor, o programa exibe a mensagem "ACHEI". Caso contrário, exibe "NÃO ACHEI". O programa realiza essa verificação por meio de uma busca sequencial dentro do vetor de nomes.

Exercício 5

O programa lê a quantidade de litros e o tipo de combustível escolhido pelo cliente, sendo Álcool representado pela letra 'A' e Gasolina pela letra 'G'. Com base nesses dados, o programa calcula o valor a ser pago considerando a tabela de descontos fornecida: para o Álcool, aplica-se um desconto de 3% por litro se forem vendidos até 20 litros e 5% por litro se forem vendidos mais de 20 litros; para a Gasolina, os descontos são de 4% por litro para até 20 litros e 6% por litro para mais de 20 litros. O resultado é então impresso, representando o valor final a ser pago pelo cliente.
